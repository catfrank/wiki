# MD 常用操作

## 表格
```md
|标题|标题|
|---|---|
|1|2|
|3|4|
|5|6|
|7|8|
```

|表头1|表头2|表头3|
|---|---|---|
|1|2|3|
|3|4|5|
|5|6|7|
|7|8|9|

```md
|标题|标题|
|:-:|-:|
|1|2|
|3|4|
|5|6|
|7|8|
```
-   -: 设置内容和标题栏居右对齐。
-   :- 设置内容和标题栏居左对齐。
-   :-: 设置内容和标题栏居中对齐。

|标题|标题|
|:-:|-:|
|1|2|
|3|4|
|5|6|
|7|8|
